Conceptos fundamentales
¿Qué es la esteganografía?

La esteganografía es la técnica de ocultar información dentro de un medio portador (imagen, audio, video) de forma que su existencia no sea evidente a simple vista. A diferencia de la criptografía, que cifra el contenido del mensaje, la esteganografía oculta la existencia del mensaje.

¿Qué es el estegoanálisis?

El estegoanálisis es la disciplina opuesta: busca detectar si una imagen (u otro medio) contiene información oculta. Este sistema implementa estegoanálisis estadístico (análisis de distribución LSB) y basado en Machine Learning (SRNet-lite).

Técnica LSB (Least Significant Bit)

En una imagen de 8 bits por canal, el bit menos significativo (LSB) de cada píxel contribuye apenas 1 unidad de intensidad. Modificar este bit es prácticamente imperceptible al ojo humano. Al reemplazar los LSBs de los canales R, G y B de forma secuencial, se puede insertar datos a una tasa de 3 bits por píxel.

Capacidad (bytes) = (ancho × alto × canales × bits_por_canal) / 8 − cabecera
Formato del protocolo StegoDetect

Toda imagen generada por este sistema contiene una cabecera binaria que permite identificar y extraer el payload de forma confiable:

[MAGIC 13B: "STEGODETECTv1"] [header_len 4B] [header JSON] [payload bytes]

El header JSON contiene: tipo, nombre, tamaño, SHA-256, algoritmo, canales y bits por canal.

Limitaciones importantes
  • JPG destruye el LSB. La compresión JPEG es lossy (con pérdida). Guardar una imagen stego como JPG elimina la información oculta. El sistema siempre genera imágenes de salida en formato PNG.
  • Extracción solo del sistema. Solo puede extraerse información de imágenes generadas por StegoDetect. Si la imagen viene de otra herramienta o es externa, el extractor reporta "no se encontró payload compatible".
  • El detector ML es probabilístico. Un AUC de 1.0 en el test set no garantiza perfección en todas las imágenes reales. El threshold óptimo (0.0404 en este modelo) fue calculado sobre el conjunto de validación.
  • Capacidad limitada. Una imagen de 512×512 px con 1 bit/canal en RGB tiene ~98 KB de capacidad útil. Para payloads grandes, usa imágenes más grandes o aumenta los bits por canal (reduce la invisibilidad).
  • Uso académico. Este sistema es un prototipo de investigación. Los resultados deben interpretarse como evidencia técnica, no como prueba legal.
Módulos del sistema
Detector ML — SRNet-lite

Red neuronal convolucional entrenada sobre BOSSBase v1.01. Aplica filtros SRM para amplificar residuos de ruido y clasifica cover vs stego.

Análisis LSB estadístico

Calcula histograma de bits 0/1 por canal, entropía binaria y detecta la cabecera del sistema para determinar si hay payload compatible.

Módulo LSB — Inserción

Inserta texto o archivos usando el protocolo STEGODETECTv1. Genera imagen stego PNG, CSV de posiciones y mapa visual de píxeles modificados.

Módulo LSB — Extracción

Lee los bits LSB, verifica el MAGIC, reconstruye el header y el payload. Valida integridad con SHA-256 antes de entregar el contenido.
